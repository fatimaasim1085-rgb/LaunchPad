The Product

Core Features

🧭

Smart Onboarding Wizard

5-step guided setup: skill quiz → AI-assisted profile builder → gig creation wizard → portfolio uploader → readiness score before going live.

🏪

Beginner-Only Marketplace

Starter Tier separates beginners from experts. Beginners compete among peers — not against sellers with 500+ reviews.

🤝

Mentor Matching

Every new freelancer is paired with an experienced mentor. First 3 orders are reviewed by mentor before delivery to client.

🔒

Escrow & Trust Layer

Client funds held in escrow. Money-back guarantee. Work review gate on first 5 orders. 3-step dispute resolution system.

💳

Pakistan-First Payments

JazzCash & EasyPaisa for domestic orders. PayPal & Payoneer for international. PKR withdrawals direct to mobile wallets.

📈

Growth Dashboard

Tracks orders, earnings, client satisfaction score, and a visual "Starter → Pro" progression path with milestone badges.

🎓

Learning Hub

Built-in guides, DigiSkills integration, and platform-issued skill certificates. Displayed directly on freelancer profiles.

📱

Mobile App

Full-featured Android + iOS app. In-app chat, push notifications, offline gig drafting, and Urdu language option.

🤖

AI Profile Assistant

OpenAI-powered tool that rewrites gig descriptions, suggests competitive pricing, and improves profile headlines for better discoverability.
